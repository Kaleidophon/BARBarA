src.prep.relations.relations module¶

This modules is about retrieving the names of entities and relations in the FB15k dataset. Because the entities are used with their (quite cryptic) Freebase ids, those have to be resolved.

Warning

Unfortunately, it isn’t possible anymore to use this code (July 2016), because the Freebase API is now deprecated; the whole Freebase project has been integrated into Wikidata. However, this code is still included to show the process of thow freebase where transformed into real names using the API and the MQL query language.
